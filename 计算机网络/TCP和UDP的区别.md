|          |                                                          |                                                            |
| -------- | -------------------------------------------------------- | ---------------------------------------------------------- |
|          | TCP                                                      | UDP                                                        |
| 面向连接 | 面向连接，在传输数据前需要建立连接。                     | 无连接的，在传输数据前不需要建立连接。                     |
| 可靠性   | 可靠的，通过重传机制、滑动窗口协议等保证数据可靠的传输。 | 尽最大努力交付数据。                                       |
| 实时性   | 实时性不强                                               | 传输速率高，适合告诉传输和实时性要求较高的通信和广播通信。 |
| 通信方式 | 每一条TCP连接只能是点对点的                              | UDP可以是1对1，1对多，多对以，和多对多的交互通信。         |
| 资源     | TCP对系统资源要求较多                                    | UDP相对较少。                                              |
|          |                                                          |                                                            |
